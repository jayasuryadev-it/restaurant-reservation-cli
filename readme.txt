# Restaurant Reservation & Order Management System (CLI)
"CREATED DURING LEARNING PYTHON PROGRAMMING"

## Description
A Python CLI-based application to manage restaurant reservations, customers, orders, and billing using MySQL database.

## Technologies Used
- Python
- MySQL
- SQL

## Features
- Add customers
- Book tables with availability check
- Place orders
- Generate bills with GST calculation

## How to Run
1. Install MySQL and create database
2. Update DB credentials in db.py
3. Run:
```bash
python main.py
